# cookin
Recipes but only the good stuff

Cookin is a private, self-hosted recipe application designed for myself :)

Unlike traditional recipe websites, Cookin focuses on making recipes easy to read, edit, search, and follow while cooking.

Core principles include:

- No ads!
- Ingredient list at the top
- Instructions at the bottom
- Each ingredient in the instructions have the amount of ingredients attached to it as well so no more scrolling up and down to figure out the amount of each ingredient at each step
- Search is fast and ingredient-aware.
- The interface is clean, distraction-free, and optimized for desktop and mobile.
- The application is intended for private use and is accessible securely through Tailscale.

For complete project documentation, see the `docs/` directory.
